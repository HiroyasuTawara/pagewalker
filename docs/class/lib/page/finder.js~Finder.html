    public attachFile(filePath: string, options: object): Promise source

    Attach the file given with filePath. Rather than setting it to the files attribute of the specified input element, set it as the data-file-object attribute of the input element. This is because the files attribute can not be rewritten by script

    Params:

    NameTypeAttributeDescription
    filePath string

    Specify the path of the file to be attached
    options object
    options.fileType object

    type attribute of the File (see HTML5 File API)
    options.replaceSubmitWithAjax object

    whether replace form-submit with ajax request, default is true.

    Promise

    public replaceSubmitWithAjax(callbacks: object): Promise source

    Replace the behavior when form is submitted with ajax. And The behavior is below, Send the request by including the File object specified in the data-file-object attribute of the input element of type=file in the form. When submit request get response 200:OK, call onSuccess callback with response object, by default replace document.body by response.text() and ipcMessage send on channel 'submit-ajax-done'. When submit request get response 302:Redirect, call onRedirect callback with response object, by default load the page of response.url, and pageLoaded event occur. When submit request get response neither 200:OK or 302:Redirect, call onRespoinse callback with response object, by default throw Error

    Params:

    NameTypeAttributeDescription
    callbacks object

    An object with callback with each attribute name
    callbacks.onSuccess function,string

    callback function for handling response when status 200.
    callbacks.onRedirect function,string

    callback function for handling response when status 302.
    callbacks.onResponse function,string

    callback function for handling response when status is not 200 and 302.

    Return:

    Promise
